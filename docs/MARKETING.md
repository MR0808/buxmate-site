# Buxmate Marketing Brief

Buxmate helps organisers manage messy group events where people don’t all know each other.

Instead of juggling spreadsheets, group chats, RSVPs, payments and event updates across multiple tools, Buxmate keeps everything in one private place.

- Best men
- Maids of honour
- Bucks party organisers
- Hens party organisers
- Golf weekends
- Group trips
- Fishing trips
- Family reunions
- Any private group event

Plan the chaos.
Split the cost.
Keep it private.

Key benefits:

- Private event hub
- RSVP tracking
- Guest management
- Shared costs
